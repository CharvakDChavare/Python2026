Key Takeaways:

1. A Class is a blueprint for creating objects.
2. An Object is an instance of a class.
3. The self parameter refers to the current object instance.
4. The __init__() method is used to initialize object attributes.
5. Class Properties (Class Variables) are shared among all objects of a class.
6. Instance Variables are unique to each object.
7. OOP helps create reusable, organized, and maintainable code.
