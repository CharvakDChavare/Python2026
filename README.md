Class Key Takeaways:

1. A Class is a blueprint for creating objects.
2. An Object is an instance of a class.
3. The self parameter refers to the current object instance.
4. The __init__() method is used to initialize object attributes.
5. Class Properties (Class Variables) are shared among all objects of a class.
6. Instance Variables are unique to each object.
7. OOP helps create reusable, organized, and maintainable code.

Variables:
Rules for Python variables:

A variable name must start with a letter or the underscore character
A variable name cannot start with a number
A variable name can only contain alpha-numeric characters and underscores (A-z, 0-9, and _ )
Variable names are case-sensitive (age, Age and AGE are three different variables)
A variable name cannot be any of the Python keywords.
