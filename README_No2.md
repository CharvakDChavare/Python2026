Python For and While Loops
Python loops are used to repeat a set of instructions multiple times.
They make programs shorter, cleaner, and easier to understand.
The two most common loops in Python are the for loop and the while loop.

For Loop
A for loop is used when you want to repeat something for each item in a group.
It works well with lists, strings, tuples, and ranges.
This loop is useful when the number of repetitions is already known.

Uses of For Loop
Reading each item from a list.

Printing numbers in a sequence.

Processing characters in a word or sentence.

Repeating a task a fixed number of times.

While Loop
A while loop is used when a task should continue until a condition becomes false.
This loop is helpful when the number of repetitions is not known before the loop starts.
It keeps running as long as the condition remains true.

Uses of While Loop
Running a program until the user enters the correct input.

Repeating tasks until a limit is reached.

Creating menus that continue until the user chooses to exit.

Performing actions based on changing conditions.

Main Difference
The main difference is simple:

A for loop is best when the number of repetitions is known.

A while loop is best when repetition depends on a condition.

Loop Control Statements
Python also provides some special statements to control loops:

break stops the loop immediately.

continue skips the current step and moves to the next one.

pass does nothing and is used as a placeholder.

Important Note
A while loop can become an infinite loop if the condition never changes.
That is why it is important to update the condition properly.
A for loop is usually safer when working with fixed data.

Best Practices
Use for loops for lists, strings, and fixed repetition.

Use while loops for condition-based repetition.

Keep loop logic simple and readable.

Avoid making loops too long or confusing.

Conclusion
for and while loops are basic and very important concepts in Python.
Understanding when to use each one helps in writing better programs.
A for loop is ideal for known repetition, while a while loop is useful for condition-based repetition.
